# Python SQLite CRUD Básico

Este repositório contém scripts de estudo que demonstram operações básicas de banco de dados usando Python e SQLite.

## Conceitos praticados

- Conexão com banco de dados
- Consultas SQL (queries)
- Uso de cursor
- Operações CRUD (Create, Read, Update, Delete)

## Tecnologias

- Python
- SQLite

## Operações implementadas

- Criar banco e tabela
- Inserir dados
- Listar dados
- Buscar registros
- Atualizar registros
- Remover registros

## Estrutura do projeto

- aula01_criar_banco.py – Criar banco e tabela  
- aula02_inserir_dados.py – Inserir registros  
- aula03_listar_alunos.py – Listar registros  
- aula04_buscar_aluno.py – Buscar registro por nome  
- aula05_atualizar_idade.py – Atualizar idade de aluno  
- aula06_remover_aluno.py – Remover registro
